CSSTreemap

When you generate CSS files with SASS and the option :line_numbers enabled, sass puts a comment like this to indicates which file/line emitted the selector.

If you paste the content of the file in the textarea, you will see a Treemap of the number of selectors/rules generated in the css per sass file

By default you will see the treemap for the AFUP's "Barometre" project.

Inspired by Franois Zaninotto talk at the PHPTour (and the IE limit of 4095 rules per CSS file).
